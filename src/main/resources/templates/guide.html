Руководство пользователя

Общая информация

Добро пожаловать в магазин мебели Comfort! Наш сайт предоставляет удобный интерфейс для просмотра и покупки мебели различных категорий.

Для всех пользователей

Просмотр товаров

  • На главной странице отображаются все доступные товары
  • Используйте меню "Категории" для фильтрации товаров по категориям
  • Нажмите на карточку товара для просмотра подробной информации

Регистрация и вход

  • Для регистрации нажмите кнопку "Регистрация" в верхнем меню
  • Заполните форму регистрации, указав email и пароль
  • После регистрации используйте кнопку "Войти" для авторизации

Для администраторов

Управление товарами

  • Перейдите в раздел "Товары" для управления ассортиментом
  • Добавляйте новые товары через форму "Добавить товар"
  • Редактируйте существующие товары, нажав кнопку "Редактировать"
  • Удаляйте товары с помощью кнопки "Удалить"

Управление категориями

  • В разделе "Категории" создавайте и управляйте категориями товаров
  • Добавляйте новые категории через форму "Добавить категорию"
  • Редактируйте названия категорий при необходимости

Управление заказами

  • Просматривайте все заказы в разделе "Заказы"
  • Создавайте новые заказы через форму "Создать заказ"
  • Используйте поиск для фильтрации заказов по дате и имени клиента
  • Отслеживайте статус заказов и управляйте ими
